Frequently Asked Questions

What is an IGNOU solved assignment?

A solved assignment is a reference answer guide prepared by experts to help students understand how IGNOU questions should be answered.

Can solved assignments be submitted directly?

No. IGNOU requires answers to be written in your own words. Solved assignments are meant only for guidance.

Where can I find solved assignments?

Students usually search using their course code and session year to download solved assignments.

What format should IGNOU assignments follow?

Assignments should include cover page details like enrollment number, program and course code.

When should assignments be submitted?

Typically assignments for June exams are submitted by April end and December exams by October end.
